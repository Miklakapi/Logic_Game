## Control
### App
- `Esc` - Back
### Game
- `W` `A` `S` `D` - Movement
### Editor
- `D` - Next stage
- `A` - Previous stage (Reset)
#### Stage 1
- `LMB` - Draw wall
- `RMB` - Erase wall
#### Stage 2
- `LMB` - Draw player
- `RMB` - Draw end point
#### Stage 3
- `LMB` - Draw active block
- `RMB` - Draw inactive block
- `W` - Next block
- `S` - Previous block
#### Stage 4
- `LMB` - Draw block
- `W` - Next block
- `S` - Previous block
#### Stage 5
- `RMB` - Take the dependency color
- `LMB` - Draw the dependency color
